> # PawfectCare – Pet Grooming Shopify Store
> A **ready-to-sell demo Shopify store** built for the **pet grooming niche**, created as part of a **2-day intensive Shopify training and hiring assessment**.
> This project demonstrates a **complete e-commerce store setup**, including branding, product collections, bundle offers, product reviews, and a fully tested **customer purchase journey** using a Shopify development store.

# Store Architecture

```
Shopify Partner Account
        │
        ▼
Development Store (PawfectCare)
        │
        ├── Theme
        │     └── Horizon 3.4.0 (customized layout & branding)
        │
        ├── Products & Collections
        │     ├── Grooming tools
        │     ├── Travel accessories
        │     └── Complete Grooming Bundle
        │
        ├── Apps & Integrations
        │     ├── Judge.me Reviews
        │     ├── Shopify Search & Discovery
        │     ├── Shopify Bundles
        │     ├── Shopify Subscriptions
        │     ├── Shopify Marketplace Connect
        │     └── Labeler – Product Labels
        │
        └── Customer Journey
              Home → Collection → Product → Cart → Checkout
```

# Home Page Design

The homepage is designed for **high conversion and user engagement**.

Features include:

* Hero banner with **primary CTA – “Shop Grooming Essentials”**
* **Shop by Category** section:

  * Grooming Essentials
  * Travel Essentials
  * Cleaning & Home Care
  * Best Sellers
* **Pet Parent Favorites** product grid
* Visible **discount pricing** with compare-at prices
* Customer **testimonials for social proof**
* Trust badges:

  * Secure Payments
  * Fast Shipping
  * Money-Back Guarantee
* Newsletter subscription for email marketing

# Product Bundling

A **Complete Pet Grooming & Care Bundle** was implemented using **Shopify Bundles**.

**Bundle Details**

| Feature       | Description                         |
| ------------- | ----------------------------------- |
| Bundle Price  | ₹4,999                              |
| Compare Price | ₹7,492                              |
| Bundle Type   | Native Shopify Bundle               |
| Inventory     | Tracked at individual product level |

Bundles help **increase average order value (AOV)**.
